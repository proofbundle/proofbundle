Dual-signature seal
dual-signature profile: seals the payload of the bundle above under profile PB-GENO-1 — ECDSA-P384 primary + Ed25519 witness, both over one SHA3-384 digest of the canonical bundle. Algorithm Registry

  
Full ProofBundle registry — 9 digest algorithms and 10 signature schemes, classical and post-quantum, all executable in this browser. Classical schemes run on WebCrypto; SHA-3 / BLAKE digests and post-quantum signatures run on the embedded audited noble implementations. Every entry carries a live round-trip test: generate, sign, verify, then confirm a flipped byte fails.

Files — Universal Artifact Custody

  
Seal any file type — binaries, PDFs, models, tarballs, images. Files are read in chunks and committed under a Merkle root (RFC 6962 style, same domain separation as the bundle tree), so large artifacts hash in constant memory and any chunk can later carry a compact inclusion proof. Nothing leaves the browser.

History

  
Everything you've sealed or checked in this browser, kept in order. Each entry links to the one before it, so if anything in the history were altered, that would be detectable. You can export this history at any time as proof of what happened and when.


  


  

    
Remote sync — optional

    

      Everything above works fully offline with no configuration. Turning this on additionally
      mirrors your local ledger entries to a server you control — it never replaces local storage,
      and every seal/verify action still succeeds instantly even if the server is unreachable.

Admissibility composition (OAA)
      
    

    
Status of the composed claim set under the ranking
      (excluded < quarantined < conjectural < declared < implementation-support < bounded-check < replay < supporting < released).
      Composition is min-rank: the weakest member governs. The Lean 4 model machine-checks commutativity, associativity,
      idempotence, absorption ('excluded'), identity ('released'), monotonicity, and the meet-semilattice law —
      11 theorems, all by exhaustive decide, typechecked under Lean 4.15.0, zero sorry.
